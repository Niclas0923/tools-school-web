### 使用教程
#### 安装和初次使用
1.  克隆或者下载整个项目
2.  下载服务器端库文件
    ```shell
    npm install
    ```
3. 进入config文件夹，对配置文件进行修改，之后返回
    ```shell
    cd config
    # 服务器相关配置
    vim server.json
    # 页面信息配置
    vim list.json
    cd ..
    ```
4. 开启服务器
    ```shell
    npm run st
    ```

#### 路由情况
- / --> /home  
  - /home
    - 主页(可转跳到其他页面)  
  - /rollCallTool
    - 点名页面
  - /weChatRelay
    - 微信接龙处理
  - /dataShow
    - 数据显示和搜索

### 开发教程
1. 先通过使用教程成功运行项目
2. 之后需要安装开发工具nodemon
    ```shell
    npm install -g nodemon
    ```
3. 进入初始文件夹运行服务器端
    ```shell
    npm run dev
    ```
4. 开启一个新的终端进入page文件夹，安装运行库，之后运行前端开发服务器
    ```shell
    cd page
    npm install
    npm run dev
    ```
5. 之后根据提示打开网页进行开发即可
